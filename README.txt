Welcome to the Instruction of our nom-nom webpage

If you want to implement this website, you need to do the following steps to set up our web page:

1. Make sure that you have installed MySQL and Python 3.0+.
2. Open your mySQL, copy and paste the code on prepare.txt to mySQL window.
3. Make sure that the SQL commands are executed successfully.
4. Open the terminal or the IDE(pychame, video studio code or any other IDE) and locate to the directory where you store app.py.
5. On the terminal or IDE, type python app.py or run the python program.
6. Copy the link that shows on the result(something like 127.0.0.1:5000).
7. Paste them on your broswer and run.
8. You are good to go.
